# Git: система контроля версий
## Презентация для ISRPO

---

## Что такое система контроля версий?
- Инструмент для отслеживания изменений
- История всех модификаций файлов
- Возможность отката к любой версии

---

## Основные понятия Git
**Репозиторий** - хранилище проекта  
**Коммит** - сохраненная версия  
**Ветка** - независимая линия разработки  
**Слияние** - объединение изменений

---

## Рабочий процесс
```bash
# Начало работы
git init
git add .
git commit -m "первый коммит"

# Работа с ветками  
git branch new-feature
git checkout new-feature
```

---

## Удаленные репозитории
- GitHub, GitLab, Bitbucket
- Синхронизация между разработчиками
- Резервное копирование кода

---

## Важность Git в разработке
- Коллаборация в команде
- Отслеживание багов
- Документирование изменений
- Профессиональный стандарт
```
